## Milestone Summary

**Key Decisions:**

- Decision: AI 模型技术选型 Python FastAPI (Rationale: Prophet/XGBoost/IsolationForest 生态成熟，微服务解耦)
- Decision: Fabric Gateway SDK 1.7.1 (Rationale: 非 legacy fabric-gateway-java 2.2.x，API 更简洁)
- Decision: Go 链码 (Rationale: Fabric 生态主流选择，性能优于 Java/Node)
- Decision: 碳核算公式参数 GB/T 32150-2015 (Rationale: 国家标准，发电 25 参数 + 电网 9 参数)
- Decision: EnterpriseAdmission 新实体不复用 EntryPermission (Rationale: 语义不同，准入证书 vs 准入权限)
- Decision: Profile 切换 fabric.enabled (Rationale: 开发用 Mock，生产用 Fabric，零代码改动切换)
- Decision: REQ-12 Fabric CA 降级为 optional (Rationale: MVP 不需要独立 CA，共享组织身份即可)
- Decision: Dual ObjectMapper snake_case/camelCase (Rationale: ML 服务用 snake_case，REST API 用 camelCase)
