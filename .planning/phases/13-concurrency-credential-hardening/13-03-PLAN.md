<objective>
Add authorization controls to all unprotected endpoints and remove the insecure header-based identity fallback.

Purpose: FileController's 10 endpoints have zero authorization -- any authenticated user can access any other user's files. The X-User-Id header fallback allows complete identity spoofing. SearchController is similarly unprotected. Prometheus endpoint exposes operational data without authentication.
Output: All endpoints require appropriate role, no header-based identity bypass, Prometheus requires auth.
</objective>
