    **oaiss-chain-ml/app/services/emission_service.py** -- `EmissionService` class:

    1. `__init__(self)` -- Set `self.model_version = "1.0.0"`

    2. `predict(self, request: EmissionForecastRequest) -> EmissionForecastResponse`:
       - Validate: `len(request.dates) == len(request.emissions)`, raise ValueError if not
       - Require minimum 2 data points; if less, raise HTTPException(422, "Insufficient data for emission forecasting: need at least 2 data points")
       - Parse dates: `dates = pd.to_datetime(request.dates)`
       - Build Prophet DataFrame: `df = pd.DataFrame({"ds": dates, "y": request.emissions})`
       - Configure Prophet for emission data (different from market data):
         - `Prophet(interval_width=0.8, changepoint_prior_scale=0.05, seasonality_mode="multiplicative")`
         - Add yearly seasonality if more than 2 years of data: `if len(dates) > 24: model.add_seasonality(name="yearly", period=365.25, fourier_order=10)`
       - Fit: `model.fit(df)`
       - Create future: `future = model.make_future_dataframe(periods=request.horizon_days, freq="D")`
       - Forecast: `forecast = model.predict(future)`
       - Extract last `horizon_days` rows: `result = forecast.tail(request.horizon_days)`
       - Extract columns: `yhat` -> forecast_emissions, `yhat_lower` -> lower_bound, `yhat_upper` -> upper_bound
       - Compute confidence: based on average relative width of confidence interval
         - `avg_width = np.mean(result["yhat_upper"] - result["yhat_lower"])`
         - `avg_emission = np.mean(result["yhat"])`
         - `confidence = max(0.0, min(1.0, 1.0 - (avg_width / avg_emission) if avg_emission > 0 else 0.5))`
       - Determine trend: same as MarketService -- compare last vs first forecast value, >2% = "up", <-2% = "down", else "stable"
       - Build `EmissionForecastResponse` with all fields
       - Log at INFO level
       - Return response

    3. Error handling: same pattern as MarketService -- catch Prophet exceptions, return 422 with detail

       a. Query historical emissions for the enterprise: `List<CarbonReport> reports = carbonReportRepository.findByEnterpriseIdAndDeletedFalse(request.getEnterpriseId())`

       c. Build feature lists from reports:

       d. Build `EmissionForecastRequest.builder().enterpriseId(request.getEnterpriseId()).dates(dates).emissions(emissions).sector(request.getSector() != null ? request.getSector() : "power_generation").horizonDays(request.getHorizonDays() != null ? request.getHorizonDays() : 180).build()`
       e. Call `EmissionForecastResponse mlResponse = mlServiceClient.predictEmission(emissionRequest)`

          - Set `enterpriseId = request.getEnterpriseId()`
       g. Return the `CarbonPredictionResponse`
