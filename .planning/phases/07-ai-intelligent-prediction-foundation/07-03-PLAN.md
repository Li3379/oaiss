    1. `__init__(self)`:
       - Set `self.model_version = "1.0.0"`

         - Generate 200 synthetic samples using `np.random` with typical ranges:
           - report_count: N(10, 3), total_emissions: N(500, 100), credit_score: N(75, 10)
           - emission_rating: N(3, 1), transaction_volume: N(200, 50), compliance_flags: N(0.5, 0.5)
           - avg_emission_per_report: N(50, 15), days_since_last_report: N(30, 15)
         - Scale with StandardScaler, fit IsolationForest: `IsolationForest(contamination=0.1, random_state=42, n_estimators=100)`
         - Store as `self._iso_model` and `self._scaler`
       - Pre-train XGBoost compliance classifier with synthetic labeled data:
         - 200 samples, label = "compliant" if credit_score > 60 and compliance_flags < 2, else "at_risk" if credit_score > 40, else "non_compliant"
         - Encode labels as 0=compliant, 1=at_risk, 2=non_compliant
         - `xgb.XGBClassifier(n_estimators=100, max_depth=3, learning_rate=0.1, objective='multi:softmax', num_class=3, random_state=42)`
         - Store as `self._xgb_model`

    2. `infer(self, request: EnterpriseInferenceRequest) -> EnterpriseInferenceResponse`:
       - Build feature vector: `[report_count, total_emissions, credit_score, emission_rating, transaction_volume, compliance_flags, avg_emission_per_report, days_since_last_report]`
       - Scale with `self._scaler.transform([features])`
       - Anomaly detection: `anomaly_score = self._iso_model.decision_function(scaled)[0]`; `is_anomaly = self._iso_model.predict(scaled)[0] == -1`
       - Compliance classification: `class_pred = self._xgb_model.predict(scaled)[0]`; map 0->COMPLIANT, 1->AT_RISK, 2->NON_COMPLIANT
       - Confidence: use `self._xgb_model.predict_proba(scaled)[0]`; take max probability as confidence
       - Risk factors -- rule-based risk factor extraction from input features:
         - If `credit_score < 50`: add "Low credit score"
         - If `compliance_flags >= 3`: add "High compliance flag count"
         - If `days_since_last_report > 90`: add "Extended reporting gap"
         - If `is_anomaly`: add "Anomalous emission pattern detected"
         - If `avg_emission_per_report > 100`: add "High average emissions per report"
       - Build and return `EnterpriseInferenceResponse`
       - Log inference result at INFO level

    3. Error handling: if feature vector construction fails, raise HTTPException(422, detail with explanation)

    2. Query carbon reports: `List<CarbonReport> reports = carbonReportRepository.findByEnterpriseIdAndDeletedFalse(enterpriseId)` -- add this method to CarbonReportRepository if not present

       - `avgEmissionPerReport = totalEmissions / reportCount`
       - `daysSinceLastReport = ChronoUnit.DAYS.between(reports.stream().map(CarbonReport::getCreatedAt).max(Comparator.naturalOrder()).orElse(LocalDateTime.now()), LocalDateTime.now())`
