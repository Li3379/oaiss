GB/T 32150-2015 Power Generation Formula:
```
E_total = E_combustion + E_desulfurization

E_combustion = SUM_i ( FC_i × NCV_i × CC_i × OF_i × 44/12 )

Where for each fuel type i:
  FC_i  = Fuel consumption (tonnes)
  NCV_i = Net calorific value (GJ/tonne)
  CC_i  = Carbon content (tonne C/GJ)
  OF_i  = Carbon oxidation rate (0-1)

E_desulfurization = (carbonate_consumed × emission_factor × conversion_rate)

25 parameters broken into:
  - 5 fuel types × 4 params each = 20 parameters (FC, NCV, CC, OF per fuel)
  - Desulfurization: carbonate_consumed, emission_factor, conversion_rate = 3 parameters
  - Report metadata: reporting_year, enterprise_name = 2 parameters
  Total = 25
```
