## Changes applied

1. `SecurityConfig.java`: 移除无效路径规则(`/enterprise/**`等)，改为统一 `anyRequest().authenticated()` + `@PreAuthorize`；Actuator 仅 health/prometheus 公开；CORS 从 `*` 收窄为可配置列表
2. `application.yml`: JWT secret 移除硬编码 fallback → `${JWT_SECRET}`；添加 `app.cors.allowed-origins` 配置
3. `application-dev.yml`: 新建，本地开发用，含 JWT secret + Flyway disabled
4. `application-docker.yml`: 改为环境变量化（DB_HOST/REDIS_HOST/JWT_SECRET）
5. `db/migration/V1__init_schema.sql`: 从 schema.sql 复制，Flyway 迁移就位
6. `docker-compose.yml`: 新建，含 MySQL + Redis + MinIO + backend + frontend 五服务编排
